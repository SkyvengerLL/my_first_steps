Grupo de viagem

Um grupo de viagem é composto por:

  • Um Capitão: o personagem jogável que representa o jogador.
  • Um Mestre das armas: o personagem que representa o grupo de marinheiros que estão alocados para combate.
  • Um Médico: o personagem responsavel pela saúde dos marinheiros.
  • Um Primeiro Imediato: responsavel pelos marinheiros alocados para outras funcionalidades, que não são combate.
  • Um Conselheiro: o repesentante do rei que é dono do barco.
  • Marinheiros: a grande maroira dos personagens.

Isso é um jogo

Isso é apenas um jogo e não representa a realidade.
