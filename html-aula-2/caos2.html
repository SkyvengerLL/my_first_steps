Caos na Antártida

Introdução


Caos na Antártida é uma aventura de sobrevivência de terror, que usa o sistema Genesys, ela é inspirada no filme The Thing (1982) com uma reviravolta de Watchmen.

A aventura foi progetada para ter três finais diferentes, dependendo da performace do grupo. É função do Narrador decidir se será possível que esses finais sejam alcançados pelos jogadores e sua jornada até lá.


Este artigo é paa quem?

Este artigo é para os olhos do Narrador de sua aventura Genesys, ele irá apresentar carreiras, itens, arquétipos e uma lista de personagens deste cenário.


O que é necessário para Narrar essa aventura?

Será necessário, este artigo, o livro " Genesys Core Rulebook" e os dados do sistema, caso não tenha dados use um rolador virtual.



Criação de personagens


Arquétipos

Os arquétipos válidos, para essa aventura, são todos os arquétipos apresentados no "Genesys Core Rulebook", que são válidos para o cenário tempos modernos (Genesys Core Rulebook pg. 164). Além desses, os seguintes arquétipostambém são válidos:


Mentalista

  • Físico: 1
  • Agilidade: 2
  • Intelecto: 2
  • Esperteza: 3
  • Vontade: 2
  • Presença: 2

Barra de Vida inicial: 10 + Físico

Barra de Estresse inicial: 10 + Vontade

Experiência inicial: 100

Perícias Iniciais: Você começa com 1 ponto em Disciplina.

Sexto Sentido: Você pode gastar 1 ponto de história por seção, para verificar se uma ação irá ajudar o grupo ou não.


Teimoso

  • Físico: 2
  • Agilidade: 2
  • Intelecto: 2
  • Esperteza: 2
  • Vontade: 3
  • Presença: 1

Barra de Vida inicial: 10 + Físico

Barra de Estresse inicial: 10 + Vontade

Experiência inicial: 100

Perícias Iniciais: Você começa com 1 ponto em Coerção.

Marcado: Durante todo encontro você pode marcar um inimigo para focar, contra ele você terá dois dados azul, em compensação todos os outros inimigos vc terá um dado negro enquanto o inimigo marcado estiver vivo.
